# GeometryPython

### Проблема:
Очень малое количество интересных игр, написанных на Pygame

### Цель: 
Создать игру-порт geometry dash, но на Python

### Средства и способы реализации: 
Язык программирования: Python

Основа: Pygame

### Технические особенности и фишки:
    
- Несколько уровней
- Рейтинг игроков
- Несколько окон
- Несколько уровней
- Анимации
- Музыка
